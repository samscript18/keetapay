# Keeta Pay

Keeta Pay is a full-stack demo for the Keeta Vibe Coding Challenge. It lets people sign in, claim a username, and send KTA on Keeta testnet using social handles instead of wallet addresses. The app keeps payments fast and human, with a live activity vibe built in.

- SDK used: `@keetanetwork/keetanet-client`
- Network: Keeta testnet
- Tech stack: Next.js, NestJS, TypeScript, Tailwind CSS, MongoDB, Privy Auth

## Prerequisites

- Node.js v18+
- npm

## Installation

```bash
git clone https://github.com/your-username/keetapay.git
cd keetapay
npm install
```

## How to run

```bash
node index.js
```

## SDK

This project uses the Keeta SDK: https://docs.keeta.com/introduction/start-developing

## Environment

Create a `.env` file and add your seed value:

```bash
SEED=your_seed_phrase_here
```
